## 데이터분류

- Structured or unstructured
    - 구조화(키, 나이) / 구조화x(저번에 어디서 만난 사람이다.)
- i.d.d. dataor non-i.i.d data
    - i.d.d : independent and indentically distributed
    - 동일한 확률분포를 가진 데이터를 수집 (독립적)
    - 동일한 확률분포란 주사위 돌리기, 동전 던지기같이 같은 확률분포를 가지는 것
- Vectorial or non-vectorial data
    - 주로 머신러닝, 딥러닝 등에서 벡터 형태로 데이터가 정리되어 있는 것
- Labeled or unlabeled data
    - labeled : 개나 고양이 사진과 이름을 알려주고 분류함
    - unlabeled : 개인지 고양이인지 모르지만 분석을 해보니까 두 분류로 나누어지고 결론적으로 각각 고양이와 개사진이었다.

## 데이터 분석방법

### Regression(회귀분석)

- cf) 상관분석 : 두 변수 사이에 상관관계가 존재하는지 그래프로 확인
    - 상관계수 : 상관관계에 대한 결론을 수치로 나타낸다.
        
        → 상관계수에 따라 상관관계가 어떠한지 알 수 있음
    
    
    → 상관분석은 오직 두 변수 사이의 상관관계만 따지고
    
    회귀분석은 두 변수 사이에 상관관계 뿐만 아니라 함수를 구하기 때문에 인과관계도 따진다.
    
- 회귀분석
    - 두 변수에 대한 함수를 구하는 것 : x(독립변수)가 얼마일 때 y(종속변수)의 값을 예측
    - 연속적인 값(data)을 가짐
        
      
    
    - 데이터 값과 비슷한 함수를 구함
    - 새로운 데이터에 대한 예측을 얼마나 잘 하는지 중요
    - overfitting되지 않게 조심해야 함
        - overfitting이란? 학습 모델이 주어진 데이터에 너무 과하게 맞춰져서 조금이라도 다른 데이터만 들어와도 다른 결과로 예측하여 결과적으로 정확도가 낮아지는 현상

### Classification(분류)

- 데이터 x에 대한 결과 y(분류값)를 통해 둘 사이의 분류 함수 f(x)를 학습
- labeled data를 사용
- 불연속적인 값을 가짐
    - 분류 값이 0 아니면 1, 아니면 어떤 값 : 함수에 대한 중간값 같은 게 없음
    
        

### Clustering(군집화)

- 데이터 사이의 숨겨진 구조를 밝혀 비슷한 데이터들을 군집화
- unlabeled data & unsupervised learning한 데이터들을 군집화
- 군집화 된 데이터 안에서 차이가 최소가 되도록 해야 함.


## AI

- AI : 사람의 행동을 따라하게 하는 것
- 머신러닝 : 데이터를 통해 (experience) 기계를 학습시키는 것
- 딥러닝 : 머신러닝 중 사람의 신경망(뉴런)을 따라해서 학습시키는 방법

## Machine learning

- 전통적인 프로그래밍과 머신러닝의 차이

- 전통적 : 사람이 프로그램을 짜서 컴퓨터에게 데이터를 보내 컴퓨터가 결과를 냄
- 머신러닝 : 컴퓨터가 데이터를 가지고 분석하는 적절한 모델을 만들어 새로운 데이터를 만들어 컴퓨터에게 보내 컴퓨터가 결과를 낸다.
- 여러가지 데이터를 가지고 분류할 때 데이터가 더 많아야 기계가 그 데이터에 대한 에측 모델을 더 정확하게 만들 수 있음
- 머신러닝
    - Training(learning) : 학습된 데이터를 가지고 모델을 통해 수행시키고 에러가 나온다면 다시 업데이트 해서 계속 모델을 학습시켜 정확도를 높임
    - Test(ingerence): 학습된 모델을 가지고 테스트 데이터를 적용시켜 모델을 연습해보고 수행능력을 확인하는 것
        
        
- 머신러닝의 분류
    - Supervised learning(지도학습)
        - 트레이닝한 데이터와 데이터의 실제 값(labels)을 가지고 regression, classfication 할 수 있는 모델을만들어줌
    - Unsupervised learning(비지도학습)
        - 학습된 데이터만 주어지고 원하는 output이 없다
        - 즉, 결과값을 나오게 하는 게 아니라 적절한 갯수의 그룹으로 나누려고 하는 것
        - 대포적으로 데이터에 대한 숨겨진 구조나 특징을 파악하는 clustering이 있다.
    - Semi-supervised learning(준지도학습)
        - 데이터가 주어지고 아웃풋이 일부만 주어짐
        - labeled data와 unlabeled data가 모두 사용되는 머신러닝 기법
        - 데이터를 수집하는 ‘데이터 레이블링’ 작업에 소요되는 자원과 비용이 높아서 일부만 labeled해 지도학습을 시켜 다른 걸 예측하는 건데 성능이 좋음
        - 즉, labeled data로 학습된 모델을 가지고 unlabeled data에 label을 달아주는 기법 (proxy-label method)
    - Reinforcement learning(강화학습)
        - Given : Rewards from sequence of actions
        - 실제 사람이 행동을 배우는 것을 따라해서 행동에 대한 결과에 보상을 주면서 학습시키는 기법
        - 예) 자율주행, 알파고, 로봇
        - 경험을 가지고 학습
        - 액션에 대한 결과 자체를 데이터로 생각해서 새로운 액션을 만듦
            - 자전거를 타면서 경험하는 것을 데이터로 해서 자전거를 더 잘 타게 함
            - 자전거를 잘 타면 reward를 많이 주고 넘어지면 reward를 줄이는 방법으로 학습시켜 자전거를 잘 타게 함
    - Self - supervised learning (자기지도학습)
        - Self-prediction : 라벨링 되지 않은 개별 데이터 샘플에 대해 스스로 task를 정해서 다른 파트를 예측함
        - Contrastive learning: 일부 데이터를 가지고 예측하고 데이터 간의 관계를 파악해서 예측

## 모델평가


- 데이터 분석이 되었다면 모델을 만들고 테스트 해야 함.
- 바로 모두 훈련시키는 게 아니라 새로운 모델에 대한 예측도를 파악해서 훈련을 시켜야 함
- 훈련하고 바로 테스트 하는 게 아니라 학습하고 검증하는 걸 반복해서 새로운 데이터에 대해 테스트하고 이런 식으로 반복해서 한다. → 성능이 잘 나오면 새로운 데이터에 대한 예측과 분석에 적용시켜서 사용
- 훈련용(학습용, 검증용), 테스트용으로 나눔
- 데이터 분석에 대한 지표
    
    
    - 정확도 : 분류 모델이 얼마나 정확하게 분류되었는지
    - 정밀도 : 예측을 참으로 했을 때 정답도 참인 비율
    - 재현율 : 정답이 참일 때 예측도 참인 비율
    - ROC : TPR(TruePositive)에 대한 FPR(False positive)
        - y축에 TP, x축에 FP 수치를 배치해 두 수치의 균형을 살펴보는 머신러닝 평가를 위한 시각화 모델
        - ROC 커브 아래 면적인 AUC(Area Under Curve)가 1에 가까워질수록, 모델이 Y를 예측하는 정확도가 높은 모델
      
            
        
    - fairness(공정성) : 요즘 모델에 대해 정확도만 따지지 않고 공정성을 따지는 게 트랜드
        - 정보이론에서 쓰는 식을 가지고 fairness 구함
        - 이슈 : 흑인 - 흑인과 관련되면 안 좋은 결과가 더 높게 나옴 (구글, 트위터, AI판사)
