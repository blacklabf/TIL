# If

: 조건이 참인지 거짓인지 판단

: 조건에 따라 처리의 흐름을 바꾸는 것


조건의 결과가 True / False 가 나옴

코드블럭 : 콜론과 들여쓰기 4칸 들어감

조건이 참일 때만 코드블럭으로 들어감 / 거짓이면 if문 다음으로 넘어감

## else

if 조건에서 True 이면 그 밑에 코드블럭으로 가서 실행하고 바로 나가고 

조건이 False이면 else의 코드블럭으로 가서 실행하고 나간다.

## elif

if 조건에서 True 이면 그 밑에 코드블럭으로 가서 실행하고 바로 나가고

if 조건에서 False 이면 elif의 조건으로 가서 True 이면 elif의 코드블럭을 실행하고 False이면 그냥 나감

if 조건에서 True 이면 그 밑에 코드 블럭으로 가서 실행하고 바로 나가고

if 조건에서 False이면 elif의 코드블럭으로 가서 참/거짓 판별 후 true이면 그 때의 elif의 코드블럭을 실행하고 False이면 else의 코드블럭을 실행한다.

- elif는 여러개 사용 가능 / else는 마지막에 1개만 쓰던가 안 쓰던가!
