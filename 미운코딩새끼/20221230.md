# Function (함수)

: 예시) print , input , type , len

: 어떤 기능을 수행하는 코드를 모아서 이름을 붙여서 불러오면 쓸 수 있도록 하는 것 

- 내장함수(파이썬에서 제공하는 함수), 직접 만들어쓰는 함수, 불러와서쓰는 함수가 있음

- 함수의 종류
    - 내장함수 : 파이썬에서 기본으로 제공하는 함수
    - 모듈의 함수 : import 해서 가져와서 쓰는 함수
    - 사용자정의 함수 : 사용자가 직접 만들어서 쓰는 함수
        - 기본 사용자정의함수
            
            - 인자(매개변수) : 입력값
            - 인자(입력값)이랑 return(출력값, 결과값, 반환값)은 있어도 되고 없어도 되긴 함
        - 여러개의 값을 돌려받는 거 (여러개 돌려주기)
            
            - 결과값이 packing
                
            - 결과값이 unpacking
        
                
                - 함수에서 하나의 tuple로 나온 값을 my_add와 my_mul에다가 각각 unpacking해서 값을 나오게 할 수도 있다.

## Why Function? (함수를 사용하는 이유)

: 함수를 한 번 만들면 필요할 때마다 다시 사용할 수 있다.

: 코드를 쉽게 관리할 수 있다. - 비슷한 기능을 하는 함수를 만들고 싶으면 살짝 수정해서 하면 됨

: 조립해서 사용가능하다. 

: 모둘 - 함수들을 모아 놓은 거

## random

: 모듈의 한 종류

: 난수를 만들거나 무작위와 관련된 함수를 포함

### random.choice()

: random 이라는 모듈의 choice라는 함수를 쓰겠다. 

: imput으로 students라는 list가 들어감.

: list 안의 값을 한 개를 쏙 뽑아서 돌려주는 거 


### random.sample()

: 여러 개의 값을 중복없이 뽑을 수 있다.

: input이 2개 들어감 , 뽑아 올 것들이랑  갯수

### random.randint()

: 숫자의 범위를 지정해서 그 숫자중에 무작위로 한 정수를 선택하는 것
